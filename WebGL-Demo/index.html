Program Instructions:

Create New Models:
This program allows the users to create multiple cylinders, cubes, or spheres within a 3d scene.
When creating a new model, you can specify the number of sides, as well as the color.
If you don't like your models, you can delete all of them from the scene and create new ones.
You can also toggle between drawing the models' lighted polygons or just the wireframes.

Control Light Sources:
Within the scene there are also two light sources: a directional light and a point light.
You can toggle both of these on and off if you'd like.

Controlling the Camera:
The camera can be moved around the scene with these key presses:
 W - Forward
 A - Left
 S - Backward
 D - Right
 Q - Down
 E - Up
The camera can be rotated around the scene by clicking and dragging the right mouse button.
You can also zoom in and out of the scene by using the scroll wheel.

Camera Animations:
The camera can be animated in to ways with these key presses:
 C - An upward spiral that can be travelled back and forth
 N - A simple camera nod to acknowledge how cool my submission is

Model Manipulation:
Individual models can be selected with a drop down menu allowing you to perform actions on the selected model:
 Shader Mode: Swap between smooth-shading or flat-shading.
 Translate: Move the model along the x, y, or z axes.
 Rotate: Rotate the model along the x, y, or z axes.
 Scale: Scale the size of the model along the x, y, or z axes.
You can also optionally download a text file containing all of the data for the selected model.
